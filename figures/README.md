## Style Guide

- **Font**: DejaVu Sans, 8pt base (matplotlib); sans-serif (GPT Image)
- **DPI**: 300 for raster; vector PDF preferred for camera-ready
- **Colors**:
  - Low family: `#C0392B` (red)
  - Midlow family: `#E67E22` (orange)
  - High family: `#27AE60` (green)
  - Base/neutral: `#2471A3` (blue)
  - Border: `#2C3E50` (dark gray)
- **Colorblind-safe**: Use shape + color; avoid red-green only
- **CHI column width**: single = 3.33", double = 7"
- **Minimum font**: 7pt in final printed size
- **No chartjunk**: no 3D, no shadows, no decorative gradients
