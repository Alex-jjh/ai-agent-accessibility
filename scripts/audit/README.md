# scripts/audit/ — Single Audit Index

Thin wrappers around the V&V pipeline. Use this directory as the **single
discoverability point** for "what audits exist and how do I run them".
Original scripts under `scripts/{amt,stage3,visual-equiv,maintenance}/`
are unchanged; these wrappers just call them.

## One-button entry

```sh
make verify-all          # all 7 stages + paper consistency
```

## Single source of truth

All paper claims trace to `analysis/_constants.py` (in-tree authoritative
constants) and `results/key-numbers.json` (machine-readable audit output,
generated by `verify-all`).

To change a paper number:
1. Edit `analysis/_constants.py`
2. Run `make verify-all`
3. Failing assertions show every downstream location that needs update
