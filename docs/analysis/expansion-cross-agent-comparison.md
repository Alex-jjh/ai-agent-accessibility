## Summary: Cross-Architecture Accessibility Effect

### Consistent Direction

- **Text-only Claude**: low 51.4% → base 100.0% (Δ = +48.6pp)
- **Text-only Llama 4**: low 51.4% → base 91.4% (Δ = +40.0pp)
- **CUA Claude**: low 42.9% → base 85.7% (Δ = +42.9pp)

### Mechanism Differences

| Agent | Primary Failure Mechanism at Low | DOM Dependency |
|-------|----------------------------------|----------------|
| Text-only Claude | Content invisibility (broken ARIA → missing a11y tree nodes) | Direct: reads a11y tree |
| Text-only Llama 4 | Same as Claude, but weaker model amplifies effect | Direct: reads a11y tree |
| SoM Claude | Phantom bids (SoM labels persist on de-semanticized elements) | Indirect: SoM overlays depend on DOM interactive elements |
| CUA Claude | Cross-layer visual changes (link→span removes clickable affordances) | Minimal: pure coordinate-based, but functional breakage still affects |
