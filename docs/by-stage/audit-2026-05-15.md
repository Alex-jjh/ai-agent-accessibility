## D. Per-stage research conclusions

### Phase 1 — Composite Variant Study (N=1,040)

- **Established the causal relationship.** Claude × text-only collapses from
  base 93.8% to low 38.5% under composite operator bundling: binary Low-vs-rest
  Z=9.83, p<10⁻¹⁹ (chi-square); Cochran-Armitage trend Z=6.635, p<10⁻⁶.
- **Asymmetric dose-response.** Degradation costs −55.4pp; enhancement (high)
  yields only −4.6pp (NS). Once accessibility crosses a sufficiency threshold,
  additional improvement saturates.
- **Token inflation as the proximate mechanism.** Median token usage 97K (low)
  vs 40K (base), 2.4× ratio with Mann-Whitney p < 10⁻³ (paper's "<10⁻⁶" claim
  was overstated; direction + magnitude match). Extreme cases hit 608K tokens
  → context overflow → failure.
- **Pathway decomposition (Claude text vs CUA contrast).** Functional pathway
  35.4pp [15.4, 55.4]; semantic pathway 20.0pp [−13.8, 52.3]. Functional CI
  excludes 0; semantic CI crosses 0 — heuristic, not a precisely estimated
  effect.
- **➜ Phase 2 motivation:** composite cannot tell us *which* operator drives
  the 55pp collapse — needs per-operator decomposition.

### Phase 2 — Mode A Depth (N=4,056)

- **L1 landmark paradox.** Targets ~6 elements at SSIM=1.000 (visually identical),
  yet causes the largest behavioral drop (−40pp depth, −28pp breadth). Smallest
  DOM footprint, largest impact.
- **L11 link→span shows agent adaptation.** 365 DOM changes but only −1.5pp
  drop for Claude (depth set) — agent recognises click failure and switches to
  `goto()` URL construction. Cross-model: Llama 4 drops −14.6pp on the same
  operator (no fallback), proving robustness is an interaction between
  environment quality and agent capability.
- **Three-tier behavioral structure.** 21 of 26 operators land within ±3pp of
  H-baseline (neutral); only L1, L5 are individually destructive. Sparsity
  itself is a finding — most operators are individually benign.
- **GEE confirms cluster-robust significance.** Both indicator models
  (destructive vs rest; Low-family vs H-family) yield β<0 with z<−2.0 after
  task-level clustering — operator effect persists after accounting for task
  random intercepts.
- **➜ Phase 3 motivation:** if 21 operators are individually benign yet their
  composite produces 55pp collapse, the missing piece is **interaction**.

### Phase 3 — C.2 Compositional Study (N=2,184)

- **15/28 pairs super-additive** (binomial test against symmetric additivity:
  p=0.019). Operator interactions amplify, not saturate.
- **L11 is an "amplifier"**: zero individual effect (+2.3pp breadth) but
  +24.1pp interaction with L6, +19.0pp with L9, +13.8pp with L4. L11 removes
  the `goto()` URL fallback that makes other operators survivable.
- **L6 latent damage**: 100% individual success but super-additive with
  everything (L1+L6 = +16.4pp, L4+L6 = +13.8pp). Reason's Swiss cheese model:
  layered defense alignment.
- **L5 ceiling**: sub-additive with most operators (L1+L5 = −17pp interaction).
  Perception-action gap saturates — once agent can't click anything, additional
  semantic degradation can't make it worse.
- **➜ Phase 6 anchor:** these interaction patterns explain how composite Low's
  55pp collapse emerges from individually benign operators (non-linear
  amplification, not additive accumulation).

### Phase 4 — DOM Signature Audit (26 ops × 12 dims)

- **Per-operator DOM fingerprint.** 12 dimensions across 4 categories: D
  (DOM structure), A (Accessibility tree roles/names/state), V (Visual SSIM
  + bbox + contrast), F (Functional interactivity).
- **DOM magnitude does NOT predict behavioral drop**: Spearman ρ = 0.426,
  p ≈ 0.10 (NS) on the 16 operators with valid Fisher tests. Large changes
  (L11: 365 tag changes) can be benign; tiny changes (L1: 5.6 changes) can
  be catastrophic.
- **L1 vs L5 vs L11 contrast underwrites §5.2 alignment scatter.** L1 (5.6
  changes, SSIM=1.0, large drop), L5 (337 changes, SSIM=0.803, large drop),
  L11 (365 changes, SSIM=0.976, small drop) sit at three corners of the
  alignment plot.
- **Visual confounds excluded.** 23 of 26 operators have median SSIM ≥ 0.99 in
  aggregate — their behavioral drops cannot be attributed to visual
  misperception.
- **➜ Cross-references with Phase 2/6 behavioral signatures** to produce the
  paper's signature-alignment classification (4 quadrants: aligned-active,
  aligned-null, agent-adaptation, structural-criticality).

### Phase 5 — Smoker / Base-Solvability Gate (N=2,052)

- **684-task → 48-task funnel (7 gates).** Every deployed-app WebArena task
  filtered through (1) deployment availability, (2) strict 3/3 base solvability,
  (3) ≥3 median steps, (4) ≤25 median steps, (5) no state mutation, (6)
  non-trivial reference answer, (7) zero infrastructure failures. Each gate
  conservative (drops more, not fewer) → reported drops are lower bounds.
- **48 passing tasks unevenly distributed**: ecommerce 22, ecommerce_admin 12,
  gitlab 13, reddit 1. Postmill's high state-sensitivity (most tasks involve
  posting/voting) eliminates 113 of 114 reddit tasks at Gate 5.
- **Pre-registered 2026-05-06/07** before any Stage 3 manipulation data
  collected. Standards locked, not fitted post-hoc.
- **Mode A / Stage 3 convergence**: only **5/13** Mode A hand-picked tasks
  pass the full 7-gate (Gate 6+7-amended) check (ecom:23, ecom:26,
  gitlab:308, admin:94, admin:95). The 8 failures are the *same* tasks
  Mode A documents as controls: noise-floor (reddit:29, reddit:67),
  operator-immune (gitlab:132, ecom:24, ecom:41, ecom:188), Docker-drift
  (gitlab:293, admin:198), stochastic baseline (admin:4). Convergence
  is stronger than "10/13 pass" — two independent selection procedures
  reject identically the same 8 tasks. (An earlier shard-B-only check
  reported 10/13; superseded 2026-05-07.)
- **➜ Phase 6 input.** Provides the 48-task locked set used by Stage 3
  breadth.

### Phase 6 — Stage 3 Breadth (N=7,488) ★ primary

- **Claude 89.5% / Llama 4 67.4%** on 48-task × 26-op × 3-rep × 2-model matrix.
  3.7× the per-cell observations of Mode A → >99% power for 20pp drop
  detection.
- **4 operators significant after Holm-Bonferroni**: L1 (−28.0pp), L9 (−12.7pp),
  L5 (−11.3pp), L12 (−7.8pp marginal). 22 NS — the sparsity finding from
  Mode A replicates at scale.
- **5/5 bottom-operator overlap** with Mode A on Claude; 4/5 on Llama 4 (ML2
  swaps in for L2). Breadth confirms depth: findings aren't 13-task artifacts.
- **Cross-model L11 adaptive recovery gap reproduces**. Claude +2.3pp, Llama 4
  −14.1pp; gap of 16.4pp is virtually identical to Mode A's 16.1pp at 3.7×
  scale → robust phenomenon, not a 13-task fluke.
- **➜ Phase 4 / Phase 6 Stage 4b cross-references** generate the paper's
  alignment scatter (DOM × behavioral) and visual control argument.

### Phase 6 — Stage 4b SSIM (9,408 captures)

- **Single source of truth for visual equivalence.** 336 unique URLs from
  Stage 3 Claude traces × 28 variants (base + base2 + 26 operators) =
  exhaustive pixel-level coverage.
- **3 operators visually distinguishable** (median SSIM < 0.99): L5 (0.834),
  L6 (0.889), L11 (0.979). 23 operators visually indistinguishable in aggregate
  — their behavioral drops are not due to visual misperception.
- **Baseline noise floor < 0.001** (base vs base2 deterministic-render check)
  — leaves wide margin for true signal detection.
- **L9 narrative-vs-SSIM reconciliation**. Paper §3 places L9 (table flatten)
  in the "structural Tier 3" narrative category; aggregate SSIM = 1.000 because
  L9 only affects table-bearing pages (Magento admin reports, GitLab tree
  views) — most of 336 URLs unaffected. Narrative classification is independent
  of SSIM-magnitude claim; both correct in their respective contexts.
- **➜ Replaces CUA-based visual control argument** at substantially greater
  rigor (336 URLs × pixel-level diff vs 13 tasks × indirect CUA performance).

## E. Top-line research synthesis (≤100 words)

Phase 1 used composite variants to establish that accessibility degradation
causally collapses agent success (N=1,040, 55pp drop). Phase 2 (Mode A) decomposed
this into 26 individual operators, finding only L1 / L5 are individually
fatal — most are benign. Phase 3 (C.2) revealed 15/28 pair-wise interactions
are super-additive: L11 amplifies, L6 layers, L5 ceilings — explaining the
individual-vs-composite paradox. Phases 4 (DOM signature) and 6-Stage4b (SSIM)
ruled out "DOM magnitude" and "visual confound" as alternative explanations.
Phase 5 (smoker) pre-registered the task set; Phase 6 Stage 3 (N=7,488) confirmed
all findings at 3.7× scale with cross-model replication. **Conclusion**: accessibility
is an environmental precondition for agent performance; structural criticality and
agent-adaptation capacity (not DOM-change magnitude) are the joint mediators.

## F. Recommendations & known discrepancies

The audit surfaced two paper claims that don't reproduce on current data — both
are documented in `analysis/_constants.py` docstrings rather than silently fixed:

### F.1 Wilcoxon token-inflation p-value (paper §5.1)

| | Paper claim | Current data |
|---|---|---|
| Median tokens (low) | 97,000 | ~102,557 |
| Median tokens (base) | 40,000 | ~42,345 |
| Inflation ratio | 2.4× | ~2.4× ✓ |
| Wilcoxon p (low > base) | < 10⁻⁶ | ~1.3 × 10⁻⁴ |

Direction + magnitude match. p-value claim is overstated by ~2 orders of
magnitude — likely from earlier computation on a smaller / differently
filtered subset. Verifier loosens the threshold to log10(p) < −3 to track
the empirically supported claim.

### F.2 GEE β values (paper §5.1 footnote)

| Model | Paper β | Paper z | Current β | Current z |
|---|---:|---:|---:|---:|
| destructive (L1+L5 vs rest) | −1.35 | −5.98 | −1.95 | −2.85 |
| Low-family vs H-family | −0.49 | −2.81 | −0.77 | −2.63 |

Both models' direction (negative) and significance (p<0.05) are preserved.
Magnitudes differ — likely a different model spec (e.g. additional task-level
covariates, different reference category). Verifier asserts SIGN +
significance instead of exact value pending paper-side reconciliation.
